SEO básico

Intro

SEO é uma abreviação de Search Engine Optimization, ou Otimização para Mecanismos de Busca.

O Front-end tem grande parte no funcionamento do SEO.

Os mecanismos de busca usam a indexação para categorizar e ranquear os dados de sites e serviços que estão na rede. Esse processo de indexação pode ser otimizado com o auxílio do SEO. Quanto mais se usam e melhor se usam as técnicas de SEO, melhor ranqueado o seu site estará. Essas técnicas são empregadas de diversas formas em várias partes do seu site.

Tag

Entre as técnicas de otimização, a utilização de tags é uma das mais comuns. Ela facilita a localização do seu site pelos mecanismos de busca, tornando-o mais relevante nos resultados de busca.

Palavras-chave

Outra técnica é o uso de palavras-chave. Para se usar essa técnica, é necessário procurar uma palavra-chave que esteja com relevância de busca pelos usuários do mecanismo de busca. Esse termo pode ser incorporado ao seu site para receber ainda mais relevância nos resultados de busca.

Uma fonte de procura para termos é: https://neilpatel.com/br/ubersuggest/

Meta Tag

Também temos as Meta Tags. Elas são muito importantes e trazem uma descrição do site. 

              
<title>Bing</title>
<meta name="description" content="O Bing o ajuda a transformar informações em ações,
     tornando mais rápido e fácil passar da pesquisa à ação." />
</code>

Canonical Tag

Essa tag é usada para quando uma url tem múltiplas variáveis dentro da sua query string. Para que se evite que as queries sejam contabilizadas individualmente, o uso de uma Meta tag junta todas elas em uma só. Por exemplo:

Tags com diferentes parâmetros:

  • https://site.com.br/home?params=val1
  • https://site.com.br/home?params=val2
  • https://site.com.br/home?params=val3

Terão a seguinte Canonical Tag: 

              
<link rel=”canonical” href=’https://site.com.br/home” />

Headers

As tags de cabeçalho h1, h2, etc, são parte da base do html que todo iniciante deveria aprender o que são. Porém deve-se ter em mente que essas tags precisam ser utilizadas devidamente. Por exemplo, todo site deve ter um h1 somente. Se os outros cabeçalhos forem usados seguindo uma lógica hierárquica do conteúdo, os mecanismos de busca podem achar as informações com mais facilidade.

Imagens

As imagens comuns, criadas com a tag <img> também são um recurso para o SEO. Mas para que isso seja possível, é necessário usar a propriedade “alt” dentro da tag, apesar dessa ser uma propriedade que visa a acessibilidade. Por exemplo: 

          
<img src=”caminho-para-imagem.png” alt=”descricao-detalhada-da-imagem” />

Links Internos

Caso o seu site tenha muitas páginas, sempre que possível, é favorável ao SEO que essas páginas contenham links umas para as outras. Os mecanismos de busca mapeiam todas as páginas de um site e, caso elas se comuniquem via as tags <img>, aumenta o ranking da sua página. Por exemplo, um link em uma página aleatória que voltaria para a página “Home” do seu site: 

          
<a href=”/home.html”>Home</a>

Obs: A página home é a página que normalmente redireciona para outras páginas do site, por isso ela deve ser usada de forma que tenha todos os links internos possíveis.

Headers

Os mecanismos de busca são capazes de registrar e comparar os conteúdos de sites. Eles sabem quando um site está copiando informações de outros sites e isso leva a uma redução no ranqueamento do site. Dessa forma, evita-se ao máximo copiar e colar informações de outros sites.

Ferramentas

Existem ferramentas que auxiliam ou automatizam a implantação do SEO.

Para blog wordpress, existe o Yoast SEO. Para o google chrome, o SeoQuake e o MozBar. E, para sites, https://neilpatel.com/br/ubersuggest/, https://moz.com/ e https://ahrefs.com/pt/.
